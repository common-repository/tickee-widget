Tickee ticket sales right from your Wordpress website.

== Description ==

tickee is a free ticketing sales platform built to manage all your events and user data, in a way never done before.
It has great features, powerful apps and is easy to set up. 



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `tickee-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit your settings in Settings > tickee and right on the page or post fields
4. Place `<?php echo tickee() ?>` in your templates or [tickee] in your post or widget bar.
4b. Additionaly, you can alter your shortcode with a custom account slug: [tickee slug='charlatan']

== Frequently Asked Questions ==

= Do I need a tickee account to use the plugin? =

For your own ticket sales you do, it's free.

= Can I promote other tickee accounts on my blog? =

If you have the permission, you can use the tickee plugin with the content of other accounts.
